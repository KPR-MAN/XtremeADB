// CAPABILITIES

Everything you need
to manage Android

A complete GUI toolkit over ADB and Fastboot — from file management to system tweaks.

📲

Device Manager

Connect multiple devices simultaneously. Switch active device with one click. Auto-detects plug and unplug.

📦

App Manager

Browse, install, uninstall, force-stop, and extract APKs. Multi-select with Ctrl+Click. Force uninstall system apps.

📁

File Explorer

Browse your device filesystem. Upload, download, rename, delete. Sortable by name or type.

⚡

Fastboot Tools

Flash any partition with A/B slot support. Pick from common partitions or write a custom name.

🔌

Wireless ADB

Connect and pair wirelessly. Full Android 11+ pairing support with IP and 6-digit code.

🔧

System Tweaks

Toggle show taps, pointer location, stay awake. Change DPI and animation scales instantly.

📺

Screen Tools

Mirror your screen via Scrcpy, take screenshots, and record your screen — all from the GUI.

🔄

Backup & Restore

Full device backup and restore with a single button. Standard ADB backup format.

LC

Live Logcat

Stream live device logs in real time. Start, stop, and save logcat output directly.
